## Question 5

Un Value Object se caractérise par…

- A. une identité stable dans le temps
- B. l'absence d'identité + l'immuabilité (défini par sa valeur)
- C. le fait d'être toujours persisté en base

<!--correction-->

**Bonne réponse : B — l'absence d'identité + l'immuabilité (défini par sa valeur).**

Un Value Object n'a pas d'identité, est immuable, et deux VO de mêmes valeurs sont égaux. L'identité stable caractérise une Entity.
