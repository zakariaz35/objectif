# Étape 4 — Manipuler des données (vers pandas)

On rassemble tout : à partir de listes de dictionnaires, on filtre, on agrège, on compte — puis on découvre `pandas`, l'outil de référence du Data-Analyst.

> **Objectif de l'étape —** transformer un jeu de données brut en réponses, en Python pur d'abord, puis avec `pandas`.

## Au programme

- Un jeu de données = une liste de dictionnaires
- Filtrer, trier, agréger (somme, moyenne, comptage)
- Regrouper par catégorie (`group by` « à la main »)
- Aperçu de `pandas` : `DataFrame`, `read_csv`, sélection de colonnes
- Premiers résumés : `describe`, `groupby`

C'est le pont vers la suite du cursus BI / Data-Analyst.
