# Étape 3 — Fonctions, modules & fichiers

Pour analyser des données, il faut savoir **organiser** son code en fonctions et **lire** des fichiers, à commencer par le CSV.

> **Objectif de l'étape —** écrire des fonctions réutilisables, importer des modules, et charger un fichier CSV en mémoire.

## Au programme

- Définir une fonction : paramètres, valeur de retour
- Arguments par défaut et nommés
- Importer un module (`import`, `from ... import ...`)
- Ouvrir un fichier avec `with open(...)`
- Lire un CSV avec le module `csv`

Le CSV est le format pivot de la BI : c'est souvent par lui qu'arrive un jeu de données.
