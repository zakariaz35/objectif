# Étape 4 — Vue 3 avancé

Une fois les bases acquises, les outils qui structurent une vraie application.

> **Objectif de l'étape —** organiser une app : navigation, état partagé, et logique réutilisable.

## Au programme

- **Composables** (`useXxx`) : extraire et réutiliser la logique
- **Vue Router** : routes, params, navigation, gardes
- **Pinia** : état global propre (stores)
- Appels API et gestion async (chargement / erreurs)
- `provide` / `inject`, `Teleport`, `Suspense`
- Performance : `v-once`, `shallowRef`, découpage/lazy-loading
